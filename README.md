# 歧路旅人：大陆的霸者 - 阵容辅助工具

为游戏《歧路旅人：大陆的霸者》设计的本地阵容辅助工具，提供角色库浏览、阵容编辑保存、回合规划等功能。

## 技术栈

- **前端框架**: Vue 3 + TypeScript + Composition API
- **构建工具**: Vite 5
- **UI 组件**: Element Plus
- **状态管理**: Pinia + 持久化插件
- **样式工具**: UnoCSS
- **路由**: Vue Router 4
- **数据爬虫**: Python 3 + BeautifulSoup（脚本独立运行）
